7-Day Curriculum

Day 1: Intro to AI & Context

  • Learn what an AI model is and how they work
  • Understand 'context' in AI and inference processes
  • Explore why context matters in multi-turn conversations

Day 2: Tokens & Context Windows

  • Master tokenization and token limits (4k/8k/128k)
  • Learn how AI models process input context
  • Understand truncation and response quality impact

Day 3: Prompt Engineering

  • Explore zero-shot, few-shot, and role-based prompting
  • Learn system vs user prompts differences

Day 4: Chaining & Memory

  • Learn prompt chaining and memory simulation
  • Cover embedding-based retrieval and memory persistence

Day 5: Context Management in APIs

  • Practice with leading AI APIs
  • Manage input/output context and multi-turn logic

Day 6: Build Project

  • Develop a versatile AI bot for university queries
  • Add memory capabilities and handle edge cases

Day 7: Review & Extend

  • Review learning, test bot, and gather feedback
  • Explore deployment and advanced memory concepts
